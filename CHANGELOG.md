# Changelog

All notable changes to the AURA Framework will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-23

### Added

- **Core Framework**
  - Five value dimensions: DLA, TA, DQP, LV, OLV
  - Maturity model with four stages (Pilot, Proven, Scaled, Optimized)
  - Risk adjustment framework with four risk categories
  - NPV, IRR, and payback period calculations

- **Python Calculator Library**
  - Complete `aura_roi` package
  - Data models for projects, tasks, costs, and risks
  - Industry presets for 9 industries
  - Visualization support (matplotlib, plotly)
  - Command-line interface

- **Documentation**
  - Methodology overview and detailed guides
  - Five value dimensions documentation
  - Maturity model documentation
  - Risk adjustment documentation
  - Quickstart guide

- **Examples**
  - Customer service agent scenario
  - Document processing scenario
  - Healthcare triage scenario
  - Supply chain optimization scenario
  - Fraud detection scenario

- **Templates**
  - Business case templates
  - Executive presentation templates

- **Infrastructure**
  - GitHub issue and PR templates
  - CI/CD workflows
  - Comprehensive test suite
